Bykea is an instant delivery service in Pakistan which is currently operating in Karachi, Lahore, Rawalpindi & Islamabad. Service can be used only for with city delivery.

== Description ==

Bykea is an instant delivery service in Pakistan, currently operating in four cities Karachi, Lahore, Rawalpindi & Islamabad. Both ecommerce or non-ecommerce sellers can use Bykea to delivery their parcels within 60 mins in the cities where Bykea operates. To get your Bykea corporate account please install Bykea app from Google Play or App Store, signup with your business mobile number and contact Bykea team at info@bykea.com and update credentials in settings to start using Bykea delivery. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Please go to “Dashboard” -> “Plugins” and select "Add New"
2. Now search for "Bykea Instant Delivery" and Install the plugin
3. Your plugin is now Installed and ready to use, next step is to enter your Bykea Credentials in settings and create your pickup location in settings area.
4. Once credentials and pickup is setup in settings, you are now ready to start delivering with Bykea

== Frequently Asked Questions ==

= Where can I delivery using Bykea instant delivery? =

You can deliver your parcels instantly in Pakistani cities where Bykea currently operates. Service is currently offered in Karachi, Lahore, Rawalpindi & Islamabad

= What type of parcels can be delivered by Bykea? =

You can only deliver parcels that meets Bykea’s terms and conditions for delivery, maximum volumetric weight with or without Cash on Delivery (COD) is 10 kg. 

= How can I sign up for Bykea corporate user? =

To sign up as Bykea corporate user you can simply open and submit this from: http://bit.ly/BykeaApiSignup
